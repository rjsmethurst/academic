Galaxy Zoo PhD Student at the University of Oxford statistically investigating galaxy evolution with a large morphological dataset.

Supervisor: Chris Lintott

My PhD wouldn't be possible without the help of the public classifying galaxy images with Galaxy Zoo. Start classifying today. If you want to know more about Galaxy Zoo, here's a talk I gave about it and here's a blog post I wrote about the seven greatest discoveries from Galaxy Zoo.

Check out some of the codes I've made:

starpy - a python package for determining the best quenched star formation history to describe the observed photometry of a galaxy using stellar population synthesis models and bayesian statistics.

prefig - a python figure class to produce presentation and poster ready plots from your already existing matplotlib code
